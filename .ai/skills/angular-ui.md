## Naming

Páginas:

xxxxPage

Ex:

- UsersPage
- DashboardPage

Componentes reutilizáveis:

xxxComponent

Ex:

- SidebarComponent
- UserCardComponent

## Layout

Sempre usar TailwindCSS.

Preferir:

- flex
- gap consistente
- spacing limpo
- responsividade

## Estrutura

Todo component:

- component.ts
- component.html

## HTML

Templates limpos.

Evitar lógica complexa no HTML.

Mover lógica para TypeScript quando necessário.
